# BERT Attention Explorer

Interactive visualization tool for exploring
attention patterns inside BERT.

## Features

- Attention Heatmaps
- Token Analysis
- Attention Graphs
- Multi-Head Comparison
- PNG Export
- CSV Export

## Tech Stack

- Python
- Streamlit
- Transformers
- PyTorch
- NetworkX
- Matplotlib
- Seaborn

## Installation

pip install -r requirements.txt

streamlit run app/app.py
